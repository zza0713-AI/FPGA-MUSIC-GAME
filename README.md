# FPGA-MUSIC-GAME 🎵🕹️

基于 **FPGA（EP4CE115F23C7）** 的音乐节奏游戏  
BUAA《数字电子技术基础 / VLSI数字系统设计》课程项目

---

## 📌 项目简介

**FPGA-MUSIC-GAME** 是一个运行在 FPGA 开发板上的音乐节奏游戏。  
项目使用 **Verilog HDL** 完成硬件逻辑设计，配合 **Python 脚本**将音乐文件转换为 FPGA 可读取的谱面数据，实现：

- 音符自动下落
- 按键实时判定
- 分数累计与显示
- 难度可调的节奏游戏体验

整体设计强调 **模块化、可扩展性与硬件实时性**。

---

## 🎯 项目功能概述

- 🎵 四轨道音乐节奏玩法  
- 🧩 16×16 LED 点阵显示音符下落  
- 📊 8 位数码管显示当前得分  
- 🎮 按键进行音符击打  
- ⚙️ 拨码开关设置游戏难度  
- ⏸️ 支持开始 / 暂停 / 重置  

---

## 🛠️ 硬件平台

| 硬件 | 说明 |
|------|------|
| FPGA | Altera EP4CE115F23C7 |
| LED 点阵 | 16 × 16 |
| 数码管 | 8 位七段数码管 |
| 按键 | F1 ~ F10 |
| 拨码开关 | SW1 ~ SW16 |

---

## 🎮 按键与拨码开关说明

### 🔘 按键定义

| 按键 | 功能 |
|------|------|
| F1 | 第 1 轨道击打 |
| F2 | 第 2 轨道击打 |
| F3 | 第 3 轨道击打 |
| F4 | 第 4 轨道击打 |
| F5 | 游戏开始 / 暂停 |
| F6 | 游戏重置 |
| F7 ~ F10 | 预留 |

### 🕹️ 拨码开关

| 拨码 | 功能 |
|------|------|
| SW[3:0] | 游戏难度（数值越大，下落速度越快） |
| SW[4:15] | 预留 |
